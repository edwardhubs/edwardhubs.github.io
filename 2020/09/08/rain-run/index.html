rain-run

2020-09-08 | 情感


今天又是雨天，不，应该说这周都是雨天，这个月都是雨天。
雨天，很神奇，它把我的时间拉回到了多年以前的一个狂风暴雨的傍晚。
一直不喜欢雨天，就像不喜欢冬天一样，但那次雨天例外。

这是炎热的让人厌烦的一天，整个空气像是在发怒，分明能感觉到热气从裤脚下灌进来，热又湿。
下午在学校上完课，吃完晚饭，晚上没有课，就跟一个朋友商量好，夜游附近的公园。
大概15分钟的路程，来到公园之后，围着湖边散步。
湖边迎面杨柳依依，微风和煦，瞬间扫除了闷热，心中甚是欢喜。
这样的惬意持续了不到一个小时，白天积压的闷热天气，疯狂的开始发泄。
没有一点预备，也没有一点防备，倾盆大雨像是有人往头上倒水一般来得突然，身上的衣服已经湿透。
我跟朋友两个人四目相对，尴尬无以言表，只能苦笑。
我正举目四处寻找避雨的场所，朋友诡秘的说，要不要试试在这样的雨中狂奔？
疯了，停顿了一秒，我的肾上腺素立马飙升，有何不可？
人生难得几回狂，此时不狂何时狂？
俗话说得好，常在河边走，哪有不湿鞋，既然湿了鞋，干脆洗个澡！
我抱着无所畏惧的心态，在这样侵略式的大雨中，肆无忌惮的享受暴雨的洗礼。
豆大的雨滴，就好像在洗澡的时候，把洗浴喷头水压调到最大，狠狠的冲刷着打在身上。
大雨似乎也看出来了我们的想法，默契的配合我们，没有半点减弱的迹象。
雨水顺着脖颈留下，穿过身体，从裤脚而出，再灌进鞋底，身体和鞋都越来越重。
我们一边跑，一边看着平常熟悉的街道，房子，路灯，在大雨的洗礼下，显得格外的干净。
同样的一段路，以不同的方式经过，奇妙的呈现出了不一样的画面，这种感觉很微妙。
而我们两个，就好像青春期的少年，为了引起他人的注意，想通过特立独行的行为方式，以满足自己被关注的需求。

我想，现在的我们应该都不会再去尝试这样的无厘头的行为了。
人被关注的需求，不会随着年龄的变化而消失，只是寻找的出口变得更加多元和柔和了许多。
